• Q
    什么是优惠券？
    A
    优惠券时零食小喵平台提供给用户专享的优惠措施，下单时可直接抵扣相应的订单金额。
  • Q
    如何获得优惠券？
    A
    小喵用户可通过“活动参与，平台推送”等多种形式获得相应金额的优惠券，用于购买支付的减免。
  • Q
    如何使用优惠券？
    A
    用户可以在订单结算页面，查看结算详情时，进入“可用券”列表选择一张想使用的优惠券，抵扣相应的金额。
  • Q
    优惠券使用次数？
    A
    单笔订单只能使用一张符合条件的优惠券，且每张优惠券只能使用一次，在订单支付成功后该优惠券立即失效。
  • Q
    优惠券使用限制？
    A
    优惠券的使用时限和商品购买限额见每张优惠券上的文字描述。
  • Q
    关于申请售后的优惠券？
    A
    用户对使用优惠券支付成功后的订单申请售后时，优惠券不会返还给用户。
